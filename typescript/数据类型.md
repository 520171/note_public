# 数据类型
>1. any: 任意类型
>2. number: 包含整形和浮点型
>3. string: ```''```，```""```，``` `es6模板字符串` ```
>4. boolean: ```true```和```false```
>5. 数组: 在类型后添加```[]```，数组元素的类型和类型需要保持一致，```const arr: number[] = [1, 3, 6, 9, ]```
>6. 元组: 元组是固定长度和每个元素类型固定的数组，```const tuple: [类型1, 类型2, 类型3, ] = [值1, 值2, 值3, ]```
